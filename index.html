Bienvenido a CENFOTEC Connect

Plataforma academica para gestionar actividades universitarias, espacios, responsables, participantes e inscripciones desde un solo lugar.

Ingresar al sistema

Mision

Facilitar la gestion de actividades universitarias mediante una plataforma web clara, organizada y accesible, que permita centralizar la administracion de espacios, responsables, participantes e inscripciones.

Vision

Convertirse en una herramienta academica confiable para apoyar la organizacion de eventos universitarios, mejorando la comunicacion, el control de cupos y la experiencia de participacion dentro de la comunidad CENFOTEC.

Valores

Organizacion

Promover una gestion ordenada de actividades, espacios y registros.

Transparencia

Brindar informacion clara sobre horarios, lugares, cupos y responsables.

Accesibilidad

Facilitar el acceso a la informacion desde una interfaz simple y comprensible.

Responsabilidad

Mantener datos confiables para apoyar la toma de decisiones administrativas.

Innovacion

Aplicar tecnologia web para mejorar procesos academicos y administrativos.
