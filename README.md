# CENFOTEC CONNECT

## Project Description

CENFOTEC CONNECT is a web platform designed to facilitate the management, promotion, and participation in university activities within Universidad CENFOTEC. The system allows administrators to manage events, participants, speakers, and university spaces, strengthening communication and integration within the academic community.

# System Modules

## User Management

- Administrator registration.
- Login and authentication.
- Password recovery.
- Role management.
- User activation and deactivation.

## Activity Management

- Activity registration.
- Activity editing and cancellation.
- Capacity management.
- Search and filtering tools.

## Space Management

- University locations and spaces registration.
- Activity location assignment.

## Responsible Person Management

- Registration of speakers and moderators.
- Assignment to activities.

## Participant Management

- Public participant registration.
- Participant information management.
- Duplicate registration control.

## Registration Management

- Activity registration process.
- Registration cancellation.
- Registration tracking and filtering.
